About ResuRadar

AI-powered resume insights built to help professionals grow with data-driven feedback.

Our Story

ResuRadar is an AI-powered resume optimization platform that helps ambitious professionals understand their resume performance and improve it with real, actionable insights.

Founded and developed by Manish Kumar, a Full Stack Developer with over 4 years of experience working across the MEAN stack (MongoDB, Express, Angular, Node.js), ResuRadar was born from a simple idea: to make resume feedback intelligent, fast, and accessible to everyone.

During his career, Manish saw how talented candidates struggled to express their strengths effectively on paper. ResuRadar bridges that gap using AI — analyzing resumes just like an Applicant Tracking System (ATS) would, and giving users a clear picture of what works, what doesn't, and how to fix it.

Our Mission

We aim to empower professionals with AI-driven insights that help their resumes stand out, ensuring great talent gets the recognition it deserves.

Technology Behind ResuRadar

  • code Frontend: Angular with a clean, modern UI
  • build Backend: Node.js + Express REST API
  • storage Database: MongoDB for scalability
  • auto_awesome AI Layer: OpenAI API + custom NLP analysis
  • security Authentication: Secure Google OAuth integration
  • cloud Deployment: Full-stack hosting on Render

What's Next

ResuRadar is evolving with new features — deeper AI scoring, advanced keyword mapping, and personalized portfolio suggestions.
